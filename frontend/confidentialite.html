Politique de Confidentialité

1. Collecte des données

Nous collectons les données suivantes lors de l'inscription : email, mot de passe (hashé), et avatar (optionnel). Aucune autre donnée personnelle n'est collectée.

2. Utilisation des données

Les données sont utilisées pour :

  • Créer et gérer votre compte utilisateur
  • Stocker votre avatar sur Cloudinary
  • Améliorer l'expérience utilisateur
  • Vous contacter si nécessaire (support)

3. Partage des données

Vos données ne sont pas vendues à des tiers. Nous utilisons :

  • Cloudinary pour le stockage des images (avatars)
  • Neon.tech pour la base de données PostgreSQL
  • eBay pour le traitement des commandes (redirection)

4. Sécurité

Les mots de passe sont hashés avec bcrypt (10 rounds). Les communications sont chiffrées (HTTPS). L'accès à la base de données est sécurisé.

5. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit à l'oubli (suppression de compte)
  • Droit à la portabilité

Pour exercer ces droits, contactez : magaudouxlucas08@icloud.com

6. Cookies

Nous utilisons uniquement des cookies strictement nécessaires :

  • Session JWT (stocké en localStorage, pas cookie)
  • Préférences panier (localStorage)
  • Cookies de Netlify/Render si déployé (techniques)

Aucun cookie de tracking publicitaire n'est déposé.

7. Conservation des données

Vos données sont conservées aussi longtemps que votre compte est actif. En cas de suppression, toutes les données sont définitivement effacées (y compris les images Cloudinary).
